## Overview
This project helps a person understand their spending habits by converting raw expense data into clear charts and simple insights.

## Problem
People often track expenses but struggle to see patterns and trends in their spending.

## What the tool does
- Reads expense data from a CSV file
- Analyzes spending by category and time
- Displays charts and summaries that are easy to understand

## Input
A CSV file containing:
- Date
- Category
- Amount
- Description

## Output
- Visual charts showing spending distribution
- Simple insights about spending behavior
